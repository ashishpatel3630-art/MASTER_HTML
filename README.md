# 🌐 HTML Master Documentation

![HTML Logo](assets/images/logo.png)

A complete **HTML5 Documentation Website** built using **pure HTML only**.

This project is designed to master HTML from **beginner to advanced level** by implementing real-world website structure, semantic HTML, forms, tables, multimedia, accessibility, SEO basics, and documentation-style layouts.

---

# 📌 Project Overview

**HTML Master Documentation** is a developer-focused learning platform that provides complete HTML references, tutorials, examples, cheat sheets, and practice resources.

The goal of this project is:

- Learn HTML deeply
- Understand professional website structure
- Practice semantic HTML5
- Build a documentation-style website
- Prepare a strong foundation before CSS and JavaScript
