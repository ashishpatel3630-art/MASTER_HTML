2. Paragraph Tag

This is a paragraph tag which is used to define paragraphs in an html pages
This tag is represented by the " p " tag in html

3. Line Break Tag

" br " This is a line break tag which is used to insert a line break in an html page.

4. Horizontal line

" hr " This is a horizontal line tag which is used to insert a horizontal line in an html page.

5. Strong Tag

" strong " This is a strong tag which is used to make the text bold in an html page.

6. Italic Tag

" i " This is an italic tag which is used to make the text italic in an html page.
