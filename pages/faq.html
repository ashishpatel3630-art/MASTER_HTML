HTML Frequently Asked Questions

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

2. Is HTML a programming language?

No, HTML is not a programming language. It is a markup language used to define webpage structure.

3. What is HTML5?

HTML5 is the latest version of HTML that introduced semantic elements, multimedia support, canvas, SVG, and modern web features.

4. What is the difference between HTML and CSS?

HTML creates the structure of a webpage, while CSS is used for styling and designing the webpage.

5. What are HTML tags?

HTML tags are keywords enclosed inside angle brackets that define elements of a webpage.

6. What are semantic HTML tags?

Semantic tags clearly describe their meaning.

7. What is the purpose of DOCTYPE in HTML?

DOCTYPE tells the browser that the document uses HTML5.

8. What is the use of the meta tag?

The meta tag provides metadata about the HTML document, such as character set, viewport settings, and description for search engines.

9. What is the difference between inline and block elements in HTML?

Inline elements do not start on a new line and only take up as much width as necessary, while block elements start on a new line and take up the full width available.

10. What is the purpose of the alt attribute in HTML?

The alt attribute provides alternative text for an image if it cannot be displayed.
